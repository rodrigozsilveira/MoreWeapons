# MoreWeapons

This mod creates basic vanilla-like weapons with different caracteristics

### 1. Daggers

* Increased Attack Speed
* Bleeding Effect
* Shorter Range

### 2. Katanas

* Dash Ability
* Sane Range as Sword
* A little faster Attack Speed

### 3. Scythes

* Increased Range 
* Increased Sweep Attack hitbox
* Decreased Attack Speed
