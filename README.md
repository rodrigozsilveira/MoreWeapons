# MoreWeapons

MoreWeapons is a Fabric mod for **Minecraft 1.21+** that introduces vanilla-style weaponry. Each weapon type features different mechanics and stats.

##  Weapon Types

### 1. Daggers
*  Increased Attack Speed.
*  Bleeding Effect, deals damage over time.
*  Shorter Range.

### 2. Katanas
*  Dash Ability on right mouse button.
*  Same reach as a vanilla sword.
*  Slightly higher attack speed than a sword.

### 3. Scythes
*   Increased Range.
*   Greatly increased sweep attack hitbox.
*   Decreased attack speed.
